# Oled mode for all themes 

Dark oled mode across obsidian themes 

## Description

CSS snippet to add dark oled mode that works across obsidian themes without taking away the unique ui elements that come with the theme.

## Installation

1. Download `force-oled-dark.css` from this repo
2. Place it in your vault's `.obsidian/snippets/` folder
3. Enable it in Settings > Appearance > CSS Snippets
